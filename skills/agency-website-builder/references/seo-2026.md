# SEO 2026 Reference

## E-E-A-T Signals (Google's Core Ranking Factors)

**E**xperience · **E**xpertise · **A**uthoritativeness · **T**rustworthiness

### Implementation Checklist
- [ ] Real author bios with photos
- [ ] LocalBusiness address (Schema)
- [ ] Real customer logos (not stock)
- [ ] Linked reviews/testimonials
- [ ] HTTPS everywhere
- [ ] Clear contact data (phone, email, address)
- [ ] About page with team

## Schema.org Markup (CRITICAL)

### Required per page-type:
- **Every page:** Organization Schema
- **Homepage:** LocalBusiness + FAQPage
- **Services page:** Service Schema + OfferCatalog
- **Contact page:** ContactPage Schema
- **Subpages:** BreadcrumbList
- **Blog posts:** Article Schema

### FAQPage = +20-30% Snippet Boost!
Always implement on Homepage or dedicated FAQ-page.

See `code-snippets/schema-org-snippets.html` for ready-to-use markups.

## On-Page SEO Essentials

### Meta-Tags Template
```html
<title>HAUPT-KEYWORD — USP · BRAND</title>
<meta name="description" content="155-160 Zeichen mit Keyword + Nutzen + CTA" />
<meta name="robots" content="index, follow, max-image-preview:large" />
<link rel="canonical" href="https://www.brand.com/page" />

<!-- Open Graph -->
<meta property="og:type" content="website" />
<meta property="og:title" content="..." />
<meta property="og:description" content="..." />
<meta property="og:url" content="..." />
<meta property="og:image" content="https://www.brand.com/og.jpg" />
<meta property="og:locale" content="de_DE" />
<meta property="og:site_name" content="BRAND" />

<!-- Twitter -->
<meta name="twitter:card" content="summary_large_image" />
```

## Image SEO

```html
<img
  src="assets/img/jonathan-portrait.jpg"
  alt="Jonathan Ries, Vertrieb & Business Development bei Netzformat"
  loading="lazy"
  width="800"
  height="1000"
/>
```

- Sinnvolle Dateinamen (`jonathan-ries-netzformat.jpg` statt `IMG_2342.jpg`)
- `loading="lazy"` für Below-the-Fold
- `loading="eager"` für Hero
- Modern Formats (WebP, AVIF) wo möglich

- `defer` für JS

## Answer Engine Optimization (AEO) — NEU 2026

Für AI-Suchen (Perplexity, ChatGPT, Google AI Overviews):

1. **Klare, strukturierte Antworten** in einfacher Sprache
2. **FAQ-Format** mit echten Fragen
3. **Listen + Tabellen** (strukturierte Daten)
4. **Konkrete Zahlen** ("4.900€ ab", "+312% Traffic")
5. **Direct Quotes** von Kunden/Experten
6. **Schema.org überall** (AI versteht das besser)

## Local SEO Tips

Für Agenturen mit physischem Standort:

1. **Google Business Profile** vollständig
2. **NAP-Konsistenz** (Name, Address, Phone) überall identisch
3. **LocalBusiness Schema** mit `geo` coordinates
4. **Hours of Operation** angeben
5. **Lokal-Keywords** ("Webdesign Berlin", "SEO Prenzlauer Berg")
6. **Branchenverzeichnisse** eintragen

## Content-SEO Strategy

### Page-Types die ranken
- **Landingpages** für Hauptkeyword
- **Service-Detailseiten** für Long-Tail
- **Case-Studies** für "Best of"-Searches
- **Blog-Posts** für informelle Suchanfragen
- **FAQ-Seiten** für AI-Suche

### Content-Tipps
- Erstes Heading: Haupt-Keyword
- Erste 100 Wörter: Value Prop + Keyword
- Long-Form (1500+ Wörter) für Pillar-Pages
- Internal-Links zu verwandten Seiten
